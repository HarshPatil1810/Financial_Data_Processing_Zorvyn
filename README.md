## Tech Stack

##Frontend:  
- React  
- React Bootstrap  
- Axios  
- Recharts (charts)  

##Backend:
- .NET Core Web API   
- SQL Server
- JWT Based Auth

  
## Features
### Dashboard
- View total income, expenses, and net balance
- Category-wise and monthly trends visualization
- Recent activity table with sorting

### Transactions
- Add, edit, delete financial records
- Filter by type, category, and date
- Pagination and sorting

### User Management
- Add, edit, delete users (admin only)
- Assign roles: Admin, Analyst, Viewer
- Role-based access control

### Authentication
- JWT-based authentication
- Role-based navigation
